3. Business Problem
Built an interactive Tableau dashboard to analyze KPIs, sales, profitability, regional performance, and product category trends using the Superstore dataset.

3. Key Insights
Technology generated the highest profit.
Office Supplies had the highest sales volume.
West region contributed the largest share of sales.
Furniture requires strategic review due to comparatively weaker profitability.
Positive relationship observed between sales and profit.
